### API Base URL

The backend base URL is configured using environment variables.

Create a `.env` file in the root of the frontend project and add:

```env
VITE_BACKEND_BASE=https://full-stack-social-media-be.vercel.app
VITE_BACKEND_BASE_LOCAL=http://localhost:5000
```
